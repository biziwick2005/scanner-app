About EduReg

EduReg is a face-verified student registration and attendance management system built entirely with web technologies — no app store needed. It runs in your browser and can be installed as a PWA on any device.

Face Recognition AI-powered face detection runs entirely in the browser
Duplicate Prevention Blocks students from registering twice using the same face
QR Attendance Lecturers generate time-limited QR codes to authorise sessions
Anti-Cheat Face scan + QR verification prevents proxy attendance
Firebase Backend Real-time database — no server maintenance needed
Installable PWA Add to home screen for a native-app-like experience
Admin Dashboard View students, attendance records, export CSV reports
Works Anywhere Share one link — any device with a browser can use it
